    class Real::Vector2

    Real::Vector2

    A simple 2D vector holding x and y as Float values. Used notably by Real::Window.size= to set the window dimensions.

    Creating a vector

    v = Real::Vector2.new(800.0, 600.0)
v.x  # => 800.0
v.y  # => 600.0

    Mutating components

    v.x = 1280.0
v.y = 720.0

    Notes

    Both x and y are stored as C double internally. Passing an Integer will be coerced to Float automatically by mruby.

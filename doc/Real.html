    Real — A Ruby Game Framework

    Real is a lightweight Ruby game framework built on top of raylib, exposed through mruby bindings.

    Quick Start

    Real::Window.size  = Real::Vector2.new(800.0, 600.0)
Real::Window.title = "My Game"

Real::Window.open do
  # called every frame at 60 FPS
end
